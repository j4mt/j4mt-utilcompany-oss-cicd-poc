Looking for reports before 2012?

Currently, we can only offer online usage reports from January 2012

If you would like to see earlier usage data, please email us at emailaddress@sseairtricity.com. When emailing us, be sure to include your:

  • Company name and contact information
  • Account number
  • MPRN / GPRN
  • The type of report you would like to generate
  • Start and end date for the report
